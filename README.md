# mazomenos-website

This is my personal website to show my skills

## Tech Stack

- [Vite](https://vitejs.dev/) — fast development and build tool
- [Tailwind CSS](https://tailwindcss.com/) — utility-first CSS framework

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Preview production build

```bash
npm run preview
```
